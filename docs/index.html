BasicsLibrary

BasicLibrary is a typographic library with a slight overlap into CSS. Thanks to it you don't have to waste time with unnecessary text styling.

How install BasicLibrary

  1. Download file style.css
  2. Add style.css inside your project folder
  3. Link style.css file to every HTML page
